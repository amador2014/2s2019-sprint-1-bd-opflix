Desafios Extras – API
Mostrar todos os favoritos de um determinado usuário - POST Lancamentos/favoritos
Usuário poderá filtrar por data de lançamento - GET Lancamentos/filtrarData/{ASC} ou {DESC}
Usuário poderá filtrar por plataforma - GET Lancamentos/{plataforma}
Usuário poderá favoritar um determinado lançamento - POST Lancamentos/favoritar{lancamento}
